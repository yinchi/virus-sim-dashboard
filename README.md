## Setup

To run the dashboard:

1. Install the `uv` Python package and project manager using the [official guide](https://docs.astral.sh/uv/getting-started/installation/).
2. Clone this repository from GitHub.
3. Execute `uv run dashboard`, which will also set up the necessary virtual environment before running the dashboard, if necessary.

To see optional command line arguments, use `uv run dashboard --help`.
